1)For Running this program, please execute command 'make' in the terminal for this directory.
It will compile the HW1.cpp file.
e.g:
$make HW1
2)Then run the executive file providing three argument- name and path for the video file, name and path for the textfile and filename for the output video withoout extension 
Please do remember to input the filename "model01.obj" first and then the "commands01.txt" in sequence as command line arguments.
e.g
$./HW1 /s/chopin/a/grad/rswarade/workspace/Warping/src/images/sample_iTunes.mov /s/chopin/a/grad/rswarade/Documents/RayTracer/input.txt outputVideo

3)The program will generate outVideo.mpg as output in the current directory.
