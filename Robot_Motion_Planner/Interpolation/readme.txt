ReadMe:
1)For Running this program, please keep robot.key and object.key in the same folder where PositionInterpolation.exe and OrientationInterpolation.exe are placed.
2)For generating robot.ang file, double click on PositionInt.exe. This action will generate 'robot.ang' file in the same location.
3)For generating object.traj file, double click on OrientationInt.exe. This action will generate 'object.traj' file in the same location.
4)For accessing the source code, find the files part1.cpp and part2.cpp for Position Interpolation and Orientation Interpolation respectively.
5) Use the generated files to test animate program.
